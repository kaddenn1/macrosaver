# Founder Principles

Every design decision must answer:

"Does this help someone save money?"

If no...

Don't build it.

---

The data is the hero.

Not animations.

Not gradients.

Not marketing copy.

---

Premium.

Fast.

Simple.

Trustworthy.

---

Every page should answer one question.

"Is this worth my money?"

---

If the user has to think...

We failed.

---

Never build for ourselves.

Build for someone standing in Costco wondering
if the $74 tub is actually a good deal.

---

Everything should move the user closer to making
a confident buying decision.
